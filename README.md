# Anti-Fantasy World Cup

2026 FIFA Dünya Kupası için tersine fantasy oyunu. Klasik fantasy'nin aksine, **kötü performans gösteren oyuncular puan kazandırır**.

## Konsept
- Kaçırılan penaltı, kırmızı kart, top kaybı = artı puan
- Gol, asist = eksi puan
- Felaket maçlarda rating bonusu
- Snake draft ile lig açma, 18 kişilik squad

## Teknoloji
- Next.js (App Router) + TypeScript
- Supabase (PostgreSQL + Auth)
- Tailwind CSS
- Vercel (hosting)
- API-Football (veri kaynağı)
